# 🐵 Yuri Odilon

**`Desenvolvedor Full Stack Jr`**

Desenvolvedor Full Stack Júnior com sólida experiência na construção de aplicações web, mobile e APIs, utilizando React, TypeScript e Fastify. Minha formação em suporte e infraestrutura de TI me proporciona uma base sólida para aplicar princípios de Arquitetura de Software, SOLID e Domain-Driven Design (DDD) no desenvolvimento de soluções robustas e escaláveis.

Atualmente, sou graduando em Ciências da Computação pela Faculdade Estácio do Amazonas. Além disso, compartilho meu conhecimento de tecnologia no canal do YouTube "[Yuri Odilon](https://www.youtube.com/@solvedbykong)", onde abordo temas importantes da Computação com React, Next, Java Spring e Fastify.
